# Calculadora Tabajara 2025

## Introdução

Calculadora capaz de realizar as operações básicas de matemática de forma simples e rápida.

## Funções

- Somas
- Subtração
- Divisão
- Multiplicações
- Tabuada 
- Histórico de operações realizadas anteriormente.

## Como utilizar

1. Baixe o código fonte ou clone o repositório
2. Abra o Prompt de comando e navegue até a pasta principal.
3. Utilize o comando abaixo para restaurar as dependências do projeto.

```
dotnet restore
```

4. Em seguida, compile a solução utilizando o comando:

```
dotnet build --configuration Release
```

5. Executando o projeto em tempo real 

```
dotnet run --project Calculadora.ConsoleAPP
```

6. Para executar o arquivo compilado, navegue ate a pasta ```./Calculadora.ConsoleApp/bin/Release/net8.0/``` e execute o arquivo :

```
Calculadora.ConsoleApp
```

## Requisitos

- .NET SDK (recomendado .NET 8.0 ou superior) para compilação e execução do projeto.
